# Руководство пользователя — агенты

1. Запусти `database/sqlserver/setup-db.ps1`, чтобы создать и заполнить базу SQL Server.
2. Запусти приложение из проекта `src/PopryzhenokAgents.App`.
3. Используй верхнее поле поиска для поиска по названию агента, email или телефону.
4. Используй сортировку, чтобы упорядочить список по названию, скидке или приоритету.
5. Используй фильтр, чтобы выбрать один тип агента; пункт `Все типы` показывает все записи.
6. Нижний счётчик показывает, сколько записей отображается из найденного количества.
7. Используй кнопки страниц для перехода по списку; на одной странице отображается 10 записей.
8. Скидка рассчитывается по строкам `ProductSale` за последние 365 дней.
9. Агенты со скидкой 25% подсвечиваются светло-зелёным цветом.
10. Выдели записи и используй панель массового обновления, чтобы изменить приоритет в SQL Server.
11. Кнопки `Добавить` и `Изменить` открывают форму с полями агента и заменой логотипа.
12. Кнопка истории открывает продажи продукции выбранного агента.
13. Удаление очищает магазины и историю приоритетов, но блокируется, если у агента есть продажи.
14. Если логотип отсутствует, используется файл `resources/picture.png`.
15. Кнопка `Обновить` заново загружает записи из SQL Server после внешних изменений базы.
