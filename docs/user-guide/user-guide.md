# User guide - agents

1. Start the app from `src/PopryzhenokAgents.App`.
2. Use the top search field to search by agent name, email, or phone.
3. Use sorting to order by name, discount, or priority.
4. Use filtering to select one agent type; `Все типы` shows all records.
5. The bottom counter shows how many records are displayed from the filtered total.
6. Use page buttons to move through the list; each page shows 10 records.
7. The discount column is calculated from yearly sales amount using exam thresholds.
8. Agents with 25 percent discount are highlighted light green.
9. Select records and use the mass update panel to change priority.
10. Add/Edit/Delete buttons demonstrate record maintenance and dependency rules.
11. Use `Сбросить данные` to restore initial imported demo data before demonstration.
12. Missing logos use `resources/picture.png`.
13. SQL Server scripts are in `database/sqlserver`; run `setup-db.ps1` before demonstrating database deployment.
